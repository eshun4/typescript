Typescript
### Introduction to Typescript
Programminlg languages created at Microsoft to address some shortcomings of JavaScript. 

- Built on top of JavaScript
- Features:
  - Static Typing. e.g. You can specify variable types like string, number, boolean, etc.
  - Code Completion.
  - Refactoring
  - Shorthand Notations
- Drawbacks:
  - Compilation step involved. (transpiling to JavaScript)
  - Discipline in coding.
